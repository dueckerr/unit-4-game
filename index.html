Welcome to the Crystal Collector Game!


You have been given a random number at the start of the game called Game Number.


There are four crystals below, each with a randomly picked value. By clicking the crystal you will add a specific amount of points to your total score.


You win the game by matching your score to Game Number. You lose the game if your score goes over the Game Number


The value of the crystals is hidden from you until you click on it.


Each time when the game starts, the number to guess changes, as does the value of each crystal.
